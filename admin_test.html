Admin Dashboard Test

Click the link below to open the admin dashboard:

Open Admin Dashboard

Manual Testing Steps:

  1. Click the link above (you may need to log in as admin first)
  2. In the dashboard, click on "Sewakan Aset" in the sidebar
  3. Open the browser's Developer Tools (F12)
  4. Go to the Console tab
  5. Look for any error messages related to "Failed to load available assets"
  6. Check the Network tab to see if the API call to /rental/api/assets/available is being made

Expected Behavior:

  • The "Aset Tersedia" tab should show a table with available assets
  • No "Failed to load available assets" error should appear
  • The API call should return a 200 status with asset data

Go to: http://127.0.0.1:5000/login
